# Rotator2D API Reference
Generated: 2026-05-23

A node that rotates its parent

## Class: Rotator2D
**Inherits:** [Node](https://docs.godotengine.org/en/stable/classes/class_node.html)


### ⚙️ Inspector Variables (Exported)
| Property | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **on** | `bool` | `true` | if disabled, will turn off all rotation |
| **max_rotation** | `float` | `0` | sets a max for the rotation. Rotation will never go above this value. |
| **min_rotation** | `float` | `0` | sets a min for the rotation. Rotation will never go below this value |
| **target_rotation** | `float` | `0` | sets the target rotation from 0 to 360 for the parent object |
| **rotation_curve** | `Curve` | `null` | Lets you adjust the speed and the way in which the parent object will move towards the target rotation |
